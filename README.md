# caesar-cipher-security-project
 Caesar Cipher Security Project
 Description

Built a Caesar Cipher tool in Python to understand basic cryptography and demonstrate its vulnerability to brute-force attacks.

 - Objectives
  Implement encryption & decryption
  Understand substitution ciphers
  Analyze weaknesses in simple encryption
 - Features
   Encrypt text
   Decrypt text
   Brute force attack
   Smart attack (guessing shifts)
   File encryption
 - Concepts Used
  ASCII values (ord, chr)
  Modulo arithmetic (% 26)
  Loops & conditionals
  Basic cryptography
